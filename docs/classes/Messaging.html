    Class Messaging

    Messaging service provides access to the user's voice mail box. It is possible using this service to connect to the voice mail box, retrieve the information and the list of voice mails, and manage the mail box. Using this service requires having a TELEPHONY_ADVANCED license.

    It is possible to download a voice mail as a wav file and to delete existing messages.

      Parameters

      • mailboxId: string

        the mail box identifier given in a MailBox object

      • voicemailId: string

        the voice mail identifier given in a VoiceMessage object

      • wavPath: string | null

        an optional destination file path for the downloaded wav file

      • loginName: string | null = null

        the user login name

      Returns Promise<string | null>

      the path to the downloaded wav file on success; null otherwise.

      Example

      // Download to a specific path
      const path = await O2G.messaging.downloadVoiceMessage("mbx001", "msg42", "/tmp/voicemail.wav");

      // Let the SDK choose the destination path
      const autoPath = await O2G.messaging.downloadVoiceMessage("mbx001", "msg42", null);
      if (autoPath) {
          console.log(`Downloaded to: ${autoPath}`);
      }

    getVoiceMessages

    • getVoiceMessages(
          mailboxId: string,
          newOnly?: boolean,
          offset?: number | null,
          limit?: number | null,
          loginName?: string | null,
      ): Promise<VoiceMessage[] | null>

      Gets the list of voice messages in the specified mail box.

      If the session has been opened for a user, the `loginName` parameter is ignored, but it is mandatory if the session has been opened by an administrator.

      Parameters

      • mailboxId: string

        the mail box identifier given in a MailBox object

      • newOnly: boolean = false

        if true, returns only unread voice messages

      • offset: number | null = null

        the offset from which to start retrieving the voice message list

      • limit: number | null = null

        the maximum number of items to return

      • loginName: string | null = null

        the user login name

      Returns Promise<VoiceMessage[] | null>

      the list of VoiceMessage objects on success; null otherwise.

      Example

      // Get all voice messages in the mailbox
      const messages = await O2G.messaging.getVoiceMessages("mbx001");

      // Get only unread messages
      const newMessages = await O2G.messaging.getVoiceMessages("mbx001", true);

      // Get a page of 10 messages starting from the 20th
      const page = await O2G.messaging.getVoiceMessages("mbx001", false, 20, 10);
