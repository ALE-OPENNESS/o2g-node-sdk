    Class OperatorConfig

    OperatorConfig represents the configuration of a CCD operator.

    A CCD operator can be an agent or an supervisor. This class provides access to the operator’s type, associated pro-ACD station, group memberships, skills, and feature settings (such as headset usage, self-assignment capability, or multiline configuration).
