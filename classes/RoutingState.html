    Class RoutingState

    RoutingState represente a user routing state. A routing state is composed of four elements:

    Routing state elements
    Remote extension activation When the user is configured with a remote extension, he has the possibility to activate or deactivate this remote extension. when the remote extension is de-activated, call are not presented on the mobile device.
    Forward The user can configure a forward, on his voice mail if any or on any other number (depending on the cOmniPCX Enterprise configuration).
    Overflow The user can configure an overflow on his voice mail. If a forward is configured, it is considered prior the overflow.
    Do Not Disturb When Do Not Disturb (DND) is activated, call are not presented to the user.
