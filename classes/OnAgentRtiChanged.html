    Class OnAgentRtiChanged

    Event representing changes in the attributes of a CCD agent.

    This class is delivered whenever one or more agent attributes change. Only the attributes that have changed since the previous notification are included.

    Typical usage:

    override onAgentRtiChanged(e: OnAgentRtiChanged): void {
      const nbServedACDCalls = e.nbOfServedACDCalls;
      // ...
    }
